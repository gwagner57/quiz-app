Which of the following HTML fragments are not valid XHTML?

<!DOCTYPE html>
<html xmlns="http://www.w3.org/1999/xhtml" xml:lang="en" lang="en">
  <head>
    <meta charset="UTF-8" />
  </head>
  <body>
    <h1>Embedding SVG in HTML5</h1>
  </body>
</html>

This is not XHTML: the head element must have a title child element!


<table>
   <tr><td>Tom</td><td>€32</td></tr>
   <tr><td>Mary</td><td>€47</td></tr>
</table>

This is not XHTML: the table element must have a tbody child element!


<audio autoplay> 
  <source src="song.ogg" type="audio/ogg" />  
  <source src="song.mp3" type="audio/mpeg" /> 
</audio>

This is not XHTML: the autoplay attribute must be used with the standard attribute-value syntax autoplay="autoplay".


<dl>
  <dt>Item1</dt><dd>Definition of Item1</dd>
  <dt>Item2</dt><dd>Definition of Item2</dd>
  <dt>Item3</dt><dd>Definition of Item3</dd>
</dl>

No, this is valid XHTML.


<table>
  <tfoot>
    <tr><th colspan="2">Enrol now!</th></tr>
  </tfoot>
  <tbody>
    <tr>
      <td>The American Civil War</td>
      <td>Dr. John Wroughton</td>
    </tr>
    <tr>
      <td>Karl the Great</td>
      <td>Dr. Maria Smith</td>
    </tr>
  </tbody>
</table>

No, this is valid XHTML: the table element has a tbody element.
