Quiz Title

Recall that an HTML form is a section of a document consisting of block elements that contain controls and labels on those controls. Which of the following form elements represent correct forms?

<form>
  <div><label>ISBN: <input name="isbn" /></label></div>
  <div><label>Title: <input name="title" /></label></div>
</form>

<form>
  <div>
    <label for="isbn">ISBN: </label><input id="isbn" name="isbn" />
    <label for="title">title: </label><input id="title" name="title" />
  </div>
</form>

<form>
  <div><label for="isbn">ISBN: </label><input name="isbn" /></div>
  <div><label for="title">title: </label><input name="title" /></div>
</form>

Missing id attributes in the input elements (needed as targets of the for references).


<form>
  <div><label>ISBN: </label><input name="isbn" /></div>
  <div><label>title: </label><input name="title" /></div>
</form>

Missing id attributes in the input elements and corresponding for attributes in the label elements, needed for connecting the label with the control.


<form>
  <label>ISBN: <input name="isbn" /></label><br />
  <label>Title: <input name="title" /></label>
</form>

Labels and controls are not wrapped by block-level elements.
