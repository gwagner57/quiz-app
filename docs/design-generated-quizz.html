Quizz App Design

The quiz app is based on a Firestore Cloud database that contains multi-lingual quiz questions with answer options. The basic version of the app consists of three parts:

  1. A component that allows selecting a file in the Moodle XML question format and uploads the questions (with type="multichoice" or "shortanswer" or "numerical") to the app's Firestore DB.
  2. A component that allows selecting a quiz from the Firestore DB and then creates/renders a static HTML page containing the entire quiz data (questions with answer options)
  3. A front-end component that allows a user taking a quiz on the basis of a static HTML page. This includes providing a quiz clock and a feed-back page after the quiz has been completed and evaluated. During the quiz, a quiz performance record collects, for each multiple-choice question, a set of sequence numbers of the chosen answer options, allowing to evaluate/grade the user's quiz performance.

The basic version is to support multiple-choice questions, short answer questions and numeric answer questions. The questions of a quiz, and the answer options of a multiple-choice question, should always be presented in a random order (shuffled).

The app's default language for text items is EN.

Database Schema

The Firestore DB has the following tables ("collections"):

quizzes

  • id
  • title
  • titleTextItem (optional): a text item ID number referencing titles in other languages
  • questions: a list of question ID numbers like [302754, 308115, ...]
  • availableLanguages: a list of language codes like [DE, FR, ES]

questions

  • id
  • category
  • questionText: HTML
  • questionTextItem (optional): a text item ID number referencing question texts in other languages
  • generalFeedback: HTML
  • hasManyCorrectAnswers: Boolean
  • answerOptions
    • sequenceNo: a natural number (1, 2, ...) that identifies the answer option
    • answerText: HTML
    • answerTextItem (optional): a text item ID number referencing answer texts in other languages
    • isCorrect: Boolean
    • gradePoints (optional): [-100, 100]
      the correct answer options need to add up to 100 in the case of hasManyCorrectAnswers, and incorrect answer options may have negative points
    • feedback: HTML

text_items

  • id
  • language: ISO language code (EN, DE, FR, ES, ...)
  • text
